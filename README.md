`React`를 이용하여 H&M 사이트를 클론코딩 했습니다.


React-Router를 사용하여 상품 이미지를 클릭 시 로그인 되어 있지 않으면 로그인 페이지로, 로그인 되어 있으면 제품 상세페이지로 넘어갈 수 있도록 작업했습니다.


반응형 웹으로 제작하여 다양한 환경 속에서도 레이아웃이나 기능 등이 망가지지 않습니다.
